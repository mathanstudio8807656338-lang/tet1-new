🔑 1. GitHub நற்சான்றுகள்
GitHub → Settings → Developer settings → Personal access tokens (classic) → New token → repo ✅ மட்டும் தேர்வு செய்யவும்
📁 2. கோப்பு தேர்வு (File Path)
💡 கோப்பு பெயர் format: data/{பாடம்}_{வகுப்பு}_{பருவம்}.json
உதா: data/tamil_c5_t1.json = தமிழ் 5ஆம் வகுப்பு பருவம் 1
data/ என்று ஆரம்பிக்கவும். உதா: data/english_c3_t1.json
✏️ 3. JSON திருத்தி (Editor)
📖 4. JSON Format வழிகாட்டி
[ { "id": "tam_5_t1_l1", ← unique id (பாடம் குறியீடு) "title": "1. பாடம் பெயர்", ← பட்டியலில் காட்டும் பெயர் "material": [ ← பாடக்குறிப்பு பகுதிகள் { "title": "பகுதி தலைப்பு", "points": ["குறிப்பு 1", "குறிப்பு 2"] ← simple bullets // OR "content": "<div>...HTML...</div>" ← rich HTML } ], "quiz": [ ← வினா-விடை பட்டியல் { "q": "வினா உரை?", "opts": ["விடை அ", "விடை ஆ", "விடை இ", "விடை ஈ"], "ans": 0, ← சரியான விடையின் index (0=அ, 1=ஆ...) "expl": "விளக்கம் உரை" } ] } ]
